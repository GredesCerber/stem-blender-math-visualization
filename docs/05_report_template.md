## Аннотация

Данный проект посвящён разработке интерактивного инструмента для трёхмерной визуализации математических функций вида z = f(x, y) в профессиональном 3D-редакторе Blender с использованием технологий Geometry Nodes и Python API (bpy). В ходе работы реализованы пять видов математических поверхностей (параболоид, гиперболический параболоид, тригонометрическая волна, радиальная волна, гауссиана), разработан алгоритм дискретизации и построения полигональной сетки, проведена серия экспериментов по влиянию параметров на форму поверхности. Результатом является методическое пособие и программный комплект, который может быть использован на занятиях по математике и информатике.

**Ключевые слова:** STEM-образование, математические функции, поверхности, 3D-визуализация, Blender, Geometry Nodes, Python, bpy, функции двух переменных.
