## Слайд 5 — Математическая модель

**Заголовок:** Математическая основа

**Содержание:**

Функция двух переменных: **z = f(x, y)**

| Функция | Формула |
|---------|---------|
| Параболоид | z = x² + y² |
| Седло | z = x² − y² |
| Волна | z = A·sin(k·x)·cos(k·y) |
| Воронка | z = sin(√(x²+y²)) |
| Гауссиан | z = A·e^(−(x²+y²)/σ²) |

**Справа:** маленькое изображение 3D-поверхности (волна или воронка)

## Слайд 7 — Реализация: Python-скрипт

**Заголовок:** Python + Blender API (bpy)

**Содержание:**
- Скрипт создаёт сетку напрямую в Blender
- Выбор функции параметром `FUNCTION_NAME`
- Настройка: амплитуда, частота, разрешение, область

**Скриншот:** вкладка Scripting в Blender с запущенным скриптом + результат в 3D-вьюпорте

**Код (фрагмент):**
```python
def f(x, y):
    return AMPLITUDE * math.sin(FREQUENCY * x) \
                     * math.cos(FREQUENCY * y)
```

## Советы по оформлению

### Цветовая схема (рекомендуется)
- **Фон:** тёмно-синий (#0D1B2A) или глубокий синий (#1A1A2E)
- **Акцент 1:** голубой (#4FC3F7) — заголовки, ключевые слова
- **Акцент 2:** оранжевый (#FFB74D) — выделение, иконки
- **Текст:** белый (#FFFFFF) или светло-серый (#E0E0E0)

### Шрифты
- **Заголовки:** Montserrat Bold или Roboto Bold (22–28pt)
- **Текст:** Roboto Regular или Open Sans (14–16pt)
- **Код:** Fira Code или Courier New (12–14pt)

### Canva
- Поиск шаблона: «Dark Tech Presentation» или «STEM Project»
- Вставить все изображения из папки рендеров
- Экспорт: PDF или PPTX

### PowerPoint / Google Slides
- Использовать тему «Современная» (тёмная)
- Добавить анимацию «Появление» для тезисов (не злоупотребляйте!)
- Вставить видео/gif на слайд 8 (демонстрация ползунков)
