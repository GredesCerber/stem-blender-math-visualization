# Глава 3. Структурированный план методички

> *Хорошая методичка — это не просто «что делать», а «почему именно так».*

---

## 3.1 Концепция финального документа

Методичка — это самодостаточный документ, который позволяет **любому читателю (учителю или студенту) с нуля воспроизвести 3D-визуализацию математических поверхностей в Blender**, не обладая предварительными знаниями о 3D-редакторах.

**Ключевой принцип построения:** *сначала объясняем зачем, потом показываем как.*

Читатель должен понять мотивацию каждого шага. Только после этого следует инструкция.

---

```
Раздел 1 (зачем?) → Раздел 2 (математика) → Раздел 3 (интерфейс) →
```

## 3.5 Принципы оформления методички

1. **Одна мысль — один абзац.** Нет длинных стен текста.
2. **Каждый шаг начинается с глагола.** «Нажмите», «Создайте», «Измените».
3. **Контрольные точки.** После каждого раздела — блок «Проверьте себя» с 2–3 вопросами.
4. **Скриншоты с подписями.** Каждый скриншот имеет номер и подпись.
5. **Выделение ключевых понятий.** Термины, формулы, предупреждения — в рамках или цветных блоках.
6. **Код выделен моноширинным шрифтом.**

## 3.6 Как воспроизвести 3D-поверхности с нуля: краткая карта

```
Установить Blender (blender.org) → Открыть Blender →
→ Удалить куб (X → Delete) →
→ Add → Mesh → Grid (размер 10, subdivisions 50) →
→ Properties → Modifier → Add → Geometry Nodes →
→ В редакторе GN: создать схему (см. Раздел 4) →
→ Видеть поверхность → Менять параметры (A, k, scale) →
→ Render → Сохранить картинку
```

Каждый из этих шагов подробно расписан в Разделе 4 с иллюстрациями и объяснениями.
