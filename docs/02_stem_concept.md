# STEM-концепция проекта: Актуальность, Цели и Задачи

## 1. Что такое STEM-образование

**STEM** — аббревиатура от:
- **S**cience (Естественные науки)
- **T**echnology (Технологии)
- **E**ngineering (Инженерия)
- **M**ath (Математика)

STEM-подход объединяет эти четыре области в **единый межпредметный проект**, где знания из каждой дисциплины применяются совместно для решения реальной или учебной задачи.

> *Ключевой принцип STEM*: не «изучить тему», а **решить задачу**, применяя знания из нескольких предметов и создавая измеримый продукт.

---

## 2. Как этот проект соответствует STEM

| Компонент | Что реализовано в проекте |
|-----------|--------------------------|
| **S** — Наука | Исследование математических закономерностей: как форма поверхности зависит от параметров функции. Наблюдение явлений, гипотезы, выводы. |
| **T** — Технологии | Использование профессионального 3D-инструмента Blender, программирование на Python, работа с API (bpy). |
| **E** — Инженерия | Проектирование системы: сетка → алгоритм вычисления → визуализация → интерфейс управления параметрами. Тестирование и итерация. |
| **M** — Математика | Функции двух переменных, тригонометрия, экспоненциальные функции, аналитическая геометрия, понятие дискретизации. |

## 3. Актуальность проекта

### 3.1 Проблема в образовании

Математика — один из наиболее **абстрактных** предметов школьной и вузовской программы. Студенты часто испытывают затруднения с функциями двух переменных, поверхностями и пространственным мышлением. Традиционный учебник показывает лишь статичный рисунок на плоскости.

### 3.2 Возможности 3D-технологий

Современные инструменты 3D-моделирования (в первую очередь Blender — бесплатный и мощный) позволяют:
- **показать** форму поверхности в реальном 3D,
- **изменить параметры** и мгновенно увидеть результат,
- **вращать, масштабировать, освещать** объект,
- **создать анимацию** изменения формы.

### 3.3 Педагогическая ценность

- **Наглядность:** 3D-модель «оживляет» формулу, делая её ощутимой.
- **Вовлечённость:** студент сам управляет параметрами — это не пассивное наблюдение.
- **Межпредметность:** объединение математики, информатики, основ 3D.
- **Современность:** навыки работы с 3D, Python и алгоритмами востребованы на рынке труда.

---

## 4. Цель проекта

**Цель:** разработать интерактивный 3D-инструмент для визуализации математических функций вида z = f(x, y) в среде Blender с использованием Geometry Nodes и Python API, и создать на его основе учебно-методическое пособие для преподавания математики и информатики.

---

## 5. Задачи проекта

1. **Изучить** математическую теорию функций двух переменных и типы поверхностей.
2. **Освоить** инструменты Blender: Geometry Nodes, Python API (bpy), базовые шейдеры.
3. **Реализовать** алгоритм построения поверхности z = f(x, y) через дискретизацию и создание mesh-объекта.
4. **Создать** интерактивный режим с ползунками для изменения параметров (амплитуда, частота, масштаб).
5. **Реализовать** не менее 5 различных функций для демонстрации (параболоид, волна, воронка, седло, гауссиан).
6. **Провести** серию экспериментов: изменить параметры и зафиксировать результаты в таблице.
7. **Оформить** методическое пособие с теорией, инструкциями и шаблонами для использования на занятиях.

---

## 6. Объект и предмет исследования

- **Объект:** математические функции двух переменных и их геометрические образы (поверхности).
- **Предмет:** методы компьютерной 3D-визуализации математических функций с управляемыми параметрами.

---

## 7. Гипотеза

> Использование интерактивной 3D-визуализации функций в среде Blender позволит студентам глубже понять связь между аналитической записью функции (формулой) и её геометрическим образом (поверхностью), что повысит качество освоения темы «Функции двух переменных».

| Метод | Применение |

## 9. Продукт проекта

По итогам работы создаются следующие артефакты:

1. **Blender-файл** (`FunctionVisualizer.blend`) как целевой артефакт для демонстрации (собирается на машине с установленным Blender).
2. **Python-скрипты** (`scripts/`) — три основных рабочих скрипта с единым CLI:
   `visualize_function.py`, `generate_surface_mesh.py`, `setup_geometry_nodes_surface.py`.
3. **Методическое пособие** (можно распечатать или открыть на экране) — на основе шаблонов из `docs/`.
4. **Презентация** — структура из 10–12 слайдов для защиты проекта или урока.
5. **Прикладной модуль pathfinding** — демонстрация A*/Dijkstra на поверхности `z=f(x,y)`.
6. **Репозиторий** на GitHub — всё в одном месте, легко передать другому студенту/учителю.

## 10. Ожидаемые результаты

- Студент/учитель сможет запустить 3D-визуализацию функции **за 5–10 минут** по инструкции.
- Будет наглядно показана связь: **формула → алгоритм → 3D-поверхность**.
- Материал может быть использован на занятиях по математическому анализу, алгебре и информатике.
- Проект демонстрирует реальный STEM-подход: от математической модели до программного продукта.

---

## 11. Применимость в учебном процессе

### Для урока математики
- Показать форму поверхности вместо рисунка в учебнике.
- Задание: «Предскажи, как изменится поверхность, если увеличить A вдвое? Проверь в Blender».

### Для урока информатики
- Разобрать Python-скрипт как пример алгоритма с циклами и функциями.
- Задание: «Добавь в скрипт новую математическую функцию».

### Для STEM-занятия / проектной работы
- Полный цикл: поставить задачу → разработать модель → провести эксперимент → сделать выводы.
- Срок: 2–4 занятия или 1–2 недели.
