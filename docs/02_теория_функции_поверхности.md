# Глава 2. Теория: функции двух переменных и поверхности

> *Понять формулу — значит увидеть форму.*

---

## 2.1 Что такое функция двух переменных?

Функция одной переменной f(x) ставит в соответствие каждому числу x единственное число y. Её график — **кривая** на плоскости.

Функция двух переменных f(x, y) ставит в соответствие каждой паре чисел (x, y) единственное число z. Её график — **поверхность** в трёхмерном пространстве.

```
f : R² → R
(x, y) ↦ z = f(x, y)
```

**Интуитивно:** представьте плоскость (координатные оси X и Y). В каждой точке этой плоскости функция задаёт высоту z. Совокупность всех таких точек (x, y, z) образует поверхность.

---

## 2.2 Построение сетки (Grid) — алгоритмическая основа

Чтобы визуализировать поверхность в Blender, нужно:

1. Создать сетку точек (x, y) — **регулярный Grid**, например 100×100 точек.
2. Для каждой точки (xᵢ, yⱼ) вычислить **z = f(xᵢ, yⱼ)**.
3. Переместить каждую вершину сетки в позицию (xᵢ, yⱼ, zᵢⱼ).

Это и есть весь алгоритм — простой, но универсальный. Любую функцию двух переменных можно визуализировать этим способом.

```
Псевдокод:
для каждой вершины v сетки:
    x = v.x
    y = v.y
    z = f(x, y)       ← подставляем формулу
    v.z = z            ← устанавливаем высоту
```

## 2.3 Ключевые поверхности проекта

### 2.3.1 Эллиптический параболоид

```
z = x² + y²
```

**Форма:** «чаша» или «блюдо». Минимум в точке (0, 0, 0).

**Математические свойства:**
- Сечение плоскостью z = const — окружность.
- Сечение плоскостью y = const — парабола.
- Применение: задачи оптимизации (поиск минимума функции).

**Учебный смысл:** первая 3D-поверхность, которую стоит показать студенту. Интуитивно понятная форма, сразу видна связь с f(x) = x².

---

### 2.3.2 Гиперболический параболоид (Седло)

```
z = x² − y²
```

**Форма:** «седло» или «чипс «Принглс»». В точке (0, 0, 0) — **седловая точка** (ни максимум, ни минимум).

**Математические свойства:**
- Сечение плоскостью x = const — парабола вниз.
- Сечение плоскостью y = const — парабола вверх.
- Ключевой объект в курсе дифференциального исчисления функций нескольких переменных.

**Учебный смысл:** показывает, что «критическая точка» не обязательно экстремум. 3D-форма «седла» — идеальная иллюстрация.

---

### 2.3.3 Волновая поверхность (Рябь)

```
z = A · sin(k·x) · cos(k·y)
```

Где:
- **A** — амплитуда (высота волн),
- **k** — частота (как часто волны повторяются).

**Форма:** рябь на воде, или поверхность с периодическими «горами» и «впадинами».

**Учебный смысл:**
1. Показывает роль **параметров** в функции — изменил A → волны выше; изменил k → волны чаще.
2. Связь с **тригонометрией**.
3. Связь с физикой: волновое движение, стоячие волны.

---

### 2.3.4 Круговые волны (Волна Безселя)

```
z = sin(√(x² + y²))
```

**Форма:** концентрические кольца — как рябь от камня в воде.

**Математические свойства:**
- Радиальная симметрия: значение z зависит только от расстояния r = √(x² + y²) от центра.
- По сути это sin(r) — синус от радиуса.

**Учебный смысл:** демонстрирует **переход к полярным координатам** и понятие радиальной симметрии.

---

### 2.3.5 Гауссов колокол

```
z = e^(−(x² + y²))
```

**Форма:** «горка» с острой вершиной в центре, плавно стремящейся к нулю.

**Математические свойства:**
- Двумерное нормальное распределение.
- Широко используется в статистике, теории вероятностей, обработке сигналов.

**Учебный смысл:** показывает **функцию, убывающую в бесконечности**, связь с вероятностью и нормальным распределением.

---

### 2.3.6 Обобщённая волна с параметрами

```
z = A · sin(k·x + φ) · e^(−α·(x² + y²))
```

Где:
- **A** — амплитуда,
- **k** — частота,
- **φ** — фазовый сдвиг,
- **α** — затухание (чем больше α, тем быстрее волна «гаснет» к краям).

**Учебный смысл:** отличный пример для **исследовательского задания** — студент изучает, что делает каждый параметр.

---

## 2.4 Параметры поверхности: что можно менять?

В проекте каждая поверхность имеет набор параметров, доступных через ползунки:

| Параметр | Обозначение | Что меняет |
|---|---|---|
| Амплитуда | A | Высоту поверхности (масштаб по z) |
| Частота | k | «Частоту» колебаний |
| Масштаб сетки | scale | Размер области видимости по x и y |
| Разрешение сетки | N | Количество вершин (качество сетки) |
| Фазовый сдвиг | φ | Смещение волн относительно начала |
| Затухание | α | Скорость «угасания» функции к краям |

## 2.5 Как из формулы получается 3D-сетка: пошагово

### Шаг 1. Создание сетки

Сетка N×N вершин покрывает область [-L, L] × [-L, L]:
```
xᵢ = -L + i · (2L / (N-1)),    i = 0, 1, ..., N-1
yⱼ = -L + j · (2L / (N-1)),    j = 0, 1, ..., N-1
```

### Шаг 2. Вычисление высот

Для каждой пары (xᵢ, yⱼ):
```
zᵢⱼ = f(xᵢ, yⱼ)
```

### Шаг 3. Формирование меша

Вершины: `(xᵢ, yⱼ, zᵢⱼ)` для всех i, j.

Грани: четырёхугольники, соединяющие соседние вершины:
```
(i,j) → (i+1,j) → (i+1,j+1) → (i,j+1)
```

### Шаг 4. Визуализация

Добавить освещение, материал (цветовая карта по z), камеру.

## 2.6 Цветовая карта (Color Ramp) — дополнительный способ «читать» поверхность

В Blender к поверхности можно применить **материал**, который окрашивает вершины в зависимости от их высоты z. Это называется **псевдоцветовая карта**.

Например:
- синий → нижние значения z (впадины);
- зелёный → средние значения z;
- красный → верхние значения z (вершины).

Такая окраска используется в:
- топографических картах (карты высот);
- тепловых картах (heatmap);
- визуализации физических полей.

Это ещё один образовательный мостик: студент учится «читать» цветовую информацию так же, как топограф читает карту.

| Тема курса | Связанная поверхность | Что иллюстрирует |
|---|---|---|
| Функции двух переменных | Параболоид, волна | Определение и область значений |

| Нормальное распределение | Гауссов колокол | Вероятность и статистика |
| Полярные координаты | Круговые волны | Связь декартовых и полярных координат |
