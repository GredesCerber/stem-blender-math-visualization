# Глава 5. Эксперименты

> *Наука начинается не с ответа, а с вопроса «а что будет, если...?»*

---

## Как проводить эксперименты?

Каждый эксперимент — это:
1. **Гипотеза**: «я предполагаю, что если изменить параметр X, то форма поверхности...»
2. **Действие**: изменить параметр в Blender (ползунком или в коде)
3. **Наблюдение**: записать/зарисовать/сфотографировать результат
4. **Вывод**: подтвердилась ли гипотеза? Что узнали?

Это исследовательская позиция — именно её цель STEM-образования.

## Эксперимент 1. Параболоид: «чаша» меняет размер

### Функция
```
z = x² + y²
```

### Цель
Понять, как изменение масштаба по осям X и Y влияет на форму параболоида.

### Начальные параметры
- Scale (Geometry Nodes / код): 1.0
- Subdivisions: 100

### Задание

Измените масштаб сетки (параметр scale) и запишите результаты:

| Scale | Описание формы | Скриншот |
|---|---|---|
| 0.5 | Широкая и «пологая» чаша, малая кривизна | `assets/renders/paraboloid_A05.png` |
| 1.0 | Исходный вид | `assets/renders/paraboloid_default.png` |
| 2.0 | Более крутые стенки и более быстрый рост z | `assets/renders/paraboloid_A2.png` |
| 5.0 | Очень крутая чаша, экстремальный перепад высот | `assets/renders/paraboloid_A5.png` |

### Вопросы для размышления

1. Как меняется «крутизна» стенок чаши при увеличении scale?
2. Что произойдёт, если использовать функцию `z = 0.5·x² + 2·y²`? Чаша станет симметричной или асимметричной?
3. Где на практике встречается форма параболоида? (подсказка: параболические антенны, кастрюли, оптические зеркала)

## Эксперимент 2. Волновая поверхность: роль амплитуды

### Цель

### Вопросы для размышления

## Эксперимент 4. Седло против параболоида: критические точки

### Функции для сравнения
```
Параболоид: z = x² + y²       (минимум в (0,0))
Седло:      z = x² − y²       (седловая точка в (0,0))
```

### Цель
Визуально понять разницу между минимумом и седловой точкой.

### Задание

1. Построить параболоид. Найти точку минимума. Записать координаты.
2. Построить седло. Проверить: является ли точка (0, 0) минимумом? Максимумом?
3. Сделать скриншоты обеих поверхностей с одного ракурса.

### Сравнительная таблица

| Характеристика | Параболоид (z = x²+y²) | Седло (z = x²−y²) |
|---|---|---|
| Форма в точке (0,0) | Гладкое «дно» чаши | Перекрестие подъёма/спада (седло) |
| Является ли (0,0) минимумом? | Да | Нет |
| Является ли (0,0) максимумом? | Нет | Нет |
| Сечение плоскостью x=0 | `z = y²` (ветви вверх) | `z = -y²` (ветви вниз) |
| Сечение плоскостью y=0 | `z = x²` (ветви вверх) | `z = x²` (ветви вверх) |

### Вопросы для размышления

1. Почему точка (0, 0) на седле называется «седловой»? Откуда это название?
2. В каком направлении функция убывает от точки (0,0) на седле? В каком возрастает?
3. Как определить тип критической точки аналитически (вспомните матрицу Гессе)?

## Эксперимент 5. Гауссов колокол: «ширина» распределения

### Функция

### Цель
Исследовать, как параметр **α** влияет на «ширину» и «высоту» колокола.

### Задание

### Вопросы для размышления

1. Меняется ли значение z в точке (0,0) при изменении α?
2. Как α связан с «шириной» нормального распределения (стандартным отклонением σ)?
3. Где в жизни встречается гауссов колокол? (подсказка: статистика, физика, обработка изображений)

## Эксперимент 6 (расширенный). Создайте свою поверхность

### Цель
Применить полученные знания и создать собственную функцию.

### Задание

1. Придумать функцию f(x, y), которую вы хотите визуализировать.
2. Описать гипотезу: «я думаю, что эта функция будет выглядеть как...»
3. Реализовать в Blender (Geometry Nodes или Python).
4. Сравнить результат с гипотезой.
5. Записать наблюдения и выводы.

### Идеи для вдохновения

```
z = sin(x) + cos(y)              — сумма волн
z = x * sin(y)                   — «скрученная» плоскость
z = sin(x² + y²)                 — квадратичные круговые волны
z = x / (x² + y² + 1)           — «всплеск» с затуханием
z = |sin(x)| + |cos(y)|         — «абсолютная рябь»
z = sin(x) * sin(y) * sin(x+y)  — интерференция трёх волн
```
