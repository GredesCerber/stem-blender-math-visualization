## 🚀 БЫСТРЫЙ СТАРТ ДЛЯ НОВИЧКИ

### За 1 минуту:
```powershell
# Создать 5 blend-файлов 
cd stem-blender-math-visualization
blender --background --python scripts/create_individual_blend_files.py

# Открыть любой файл
blender FunctionVisualizer_Wave.blend &
```

**Результат:** красивая 3D-волна, которой можно крутить мышкой!
