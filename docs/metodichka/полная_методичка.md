## РАЗДЕЛ 1. ЗАЧЕМ ЭТО НУЖНО

### 1.1 Проблема: мы показываем плоские картинки объёмных понятий

Когда студент видит на доске формулу **z = sin(x)·cos(y)** и рядом двумерный рисунок, его мозг должен совершить невероятный по сложности переход: от абстрактного символа — к пространственному образу, которого он никогда не видел.

Большинство студентов этого перехода не совершают. Они просто запоминают название («параболоид», «волновая поверхность», «седловая точка») и надеются, что на экзамене не спросят ничего конкретного.

Это не проблема способностей. Это проблема инструмента.

### 1.2 Что происходит с мотивацией

Непонимание рождает тревогу. Математическая тревожность (math anxiety) — хорошо изученное психологическое явление: ощущение «математика — это не моё» закрепляется именно в тот момент, когда студент несколько раз подряд не понимает материал, не видит смысла и не получает удовлетворения от работы.

3D-визуализация напрямую решает эту проблему: студент видит результат своих действий. Он изменил формулу — поверхность изменилась. Изменил параметр — понял, что он означает. Это немедленная обратная связь, которой лишён традиционный учебник.

### 1.3 Почему Blender — правильный инструмент

Blender бесплатен, работает на любой операционной системе, используется в профессиональной индустрии (кино, игры, анимация) и имеет полноценный Python API. Это не «учебная игрушка» — это реальный профессиональный инструмент, доступный каждому.

Использование Blender на уроке математики или информатики даёт студенту:

- **визуальный опыт**: он видит трёхмерную поверхность, может смотреть на неё с любого угла
- **вычислительный опыт**: он пишет формулу в коде и видит, как алгоритм создаёт объект
- **профессиональный опыт**: он работает с инструментом, который используют в реальной индустрии

## РАЗДЕЛ 2. МАТЕМАТИЧЕСКАЯ ОСНОВА

### 2.1 Функция двух переменных

Функция двух переменных — это правило, которое каждой паре чисел (x, y) ставит в соответствие одно число z:

```
z = f(x, y)
```

График такой функции — **поверхность** в трёхмерном пространстве.

**Примеры:**
- f(x, y) = x + y — наклонная плоскость
- f(x, y) = x² + y² — параболоид (чаша)
- f(x, y) = x² − y² — гиперболический параболоид (седло)
- f(x, y) = sin(x)·cos(y) — волновая рябь

### 2.2 Пять ключевых поверхностей

#### Эллиптический параболоид
```
z = x² + y²
```
**Форма:** «чаша». Минимум в (0,0). Все значения z ≥ 0.  
**Применение:** задачи оптимизации, параболические антенны, архитектура.

#### Гиперболический параболоид (Седло)
```
z = x² − y²
```
**Форма:** «седло» или «чипс «Принглс»». Седловая точка в (0,0) — ни минимум, ни максимум.  
**Применение:** теория оптимизации, дифференциальное исчисление.

#### Волновая поверхность
```
z = A · sin(k·x) · cos(k·y)
```
**Параметры:** A — амплитуда, k — частота.  
**Форма:** рябь, «горы и впадины».  
**Применение:** физика волн, теория поля.

#### Круговые волны
```
z = sin(√(x² + y²))
```
**Форма:** концентрические кольца, как рябь от камня в воде.  
**Связь:** полярные координаты — функция sin(r), где r — расстояние от центра.

#### Гауссов колокол
```
z = e^(−(x² + y²))
```
**Форма:** горка с острой вершиной в центре.  
**Применение:** нормальное распределение, статистика, обработка сигналов.

### 2.3 Алгоритм построения поверхности

```
1. Создать сетку N×N точек (xᵢ, yⱼ) в области [-L, L] × [-L, L]
2. Для каждой точки вычислить z = f(xᵢ, yⱼ)
3. Переместить вершину в позицию (xᵢ, yⱼ, zᵢⱼ)
4. Соединить соседние вершины гранями — получаем меш
```

Именно этот алгоритм реализован и в Geometry Nodes, и в Python-скрипте.

## РАЗДЕЛ 4. GEOMETRY NODES: ИНТЕРАКТИВНЫЕ ПОВЕРХНОСТИ

### 4.1 Что такое Geometry Nodes

Geometry Nodes — это система визуального программирования в Blender. Вместо текстового кода вы соединяете «узлы» (nodes) — каждый узел выполняет одну операцию. Параметры (A, k, scale) становятся ползунками, доступными прямо в интерфейсе.

### 4.2 Пошаговое построение параболоида z = x² + y²

**Шаг 1: Создать сетку**
```
Shift+A → Mesh → Grid
В панели «Add Grid»: X/Y Size = 10, Subdivisions = 100
```

**Шаг 2: Добавить Geometry Nodes**
```
Properties → Modifier Properties → Add Modifier → Geometry Nodes → New
Переключить редактор на «Geometry Node Editor»
```

**Шаг 3: Построить схему вычисления z**
```
1. Shift+A → Input → Position           (получаем координаты вершины)
2. Shift+A → Utilities → Vector → Separate XYZ  (разбиваем на X, Y, Z)
3. Shift+A → Utilities → Math [×2]     (Multiply: x*x и y*y)
4. Shift+A → Utilities → Math           (Add: x² + y²)
5. Shift+A → Utilities → Vector → Combine XYZ  (собираем вектор 0, 0, z)
6. Shift+A → Geometry → Set Position   (применяем смещение)
```

**Шаг 4: Подключить узлы**
```
Position → Separate XYZ
X → [Math Multiply] → (x*x) → [Math Add]
Y → [Math Multiply] → (y*y) → [Math Add]
[Math Add] (результат) → [Combine XYZ: Z]
[Combine XYZ] → Set Position: Offset
Group Input: Geometry → Set Position: Geometry
Set Position → Group Output: Geometry
```

**Результат:** на экране появится параболоид.

### 4.3 Добавление параметров A и k

Чтобы добавить ползунок:
```
В Group Input — нажать «+» → Float → назвать «Amplitude» (default: 1.0)
Подключить A к нужному Math-узлу (Multiply результата на A)
```

Теперь в панели модификатора (синяя гайка) появится ползунок «Amplitude».

## РАЗДЕЛ 5. PYTHON (BPY): ПРОГРАММИРОВАНИЕ ПОВЕРХНОСТЕЙ

### 5.1 Почему стоит изучить bpy

Geometry Nodes отлично подходит для интерактивной демонстрации. Python даёт:
- полную свободу в формулах (любые математические функции)
- возможность автоматизации (цикл по функциям, экспорт данных)
- опыт программирования, востребованный в индустрии

### 5.2 Структура bpy-скриптов

В проекте три основных скрипта:

1. `scripts/visualize_function.py` — быстрый запуск + colormap + PNG.
2. `scripts/generate_surface_mesh.py` — mesh-генерация с материалом.
3. `scripts/setup_geometry_nodes_surface.py` — интерактивный GN-граф.

Единый слой параметров и формул вынесен в `scripts/function_library.py`.

### 5.3 Как изменить функцию

Скрипты используют единый CLI-интерфейс:

- `--function` (`paraboloid`, `saddle`, `wave`, `ripple`, `gaussian`, `custom`)
- `--resolution`
- `--x-min`, `--x-max`, `--y-min`, `--y-max`
- `--amplitude`, `--frequency`, `--sigma`

Чтобы добавить новую формулу, расширьте реестр функций в `scripts/function_library.py`.

### 5.4 Запуск скрипта

В Blender:
```
Переключить рабочую область на «Scripting»
Нажать «Open» → выбрать scripts/visualize_function.py
Нажать ▶ (Run Script)
```

Из командной строки:
```powershell
blender --background --python scripts\visualize_function.py -- --function wave --resolution 100 --amplitude 2 --frequency 3 --output assets\renders\wave_A2_k3.png
```
