# Пошаговые инструкции по реализации в Blender

## Предварительные требования

- **Blender 3.6 или выше** — скачать бесплатно: https://www.blender.org/download/
- **VS Code** (опционально) — для редактирования Python-скриптов вне Blender
- Базовое знакомство с интерфейсом Blender (можно пройти за 30 минут по официальному туториалу)

---

## Часть 1: Python-скрипт (generate_surface_mesh.py)

Это **самый быстрый способ** получить поверхность — скрипт создаёт полигональный объект прямо в сцене.

### Шаг 1: Открыть Blender и перейти в Scripting

1. Запустите Blender.
2. В верхней части экрана найдите строку вкладок: `Layout | Modeling | Sculpting | ... | Scripting`.
3. Нажмите **Scripting**.
4. Слева откроется текстовый редактор, справа — 3D-вьюпорт.

### Шаг 2: Загрузить скрипт

**Вариант A — из файла:**
1. В текстовом редакторе нажмите **Open** (вверху редактора).
2. Найдите файл `scripts/generate_surface_mesh.py` и откройте его.

**Вариант B — вставить вручную:**
1. Нажмите **New** в редакторе.
2. Скопируйте содержимое `generate_surface_mesh.py` и вставьте в редактор (`Ctrl+V`).

### Шаг 3: Настроить параметры (при необходимости)

Скрипт поддерживает единые CLI-параметры:

```powershell
blender --background --python scripts\generate_surface_mesh.py -- --function wave --resolution 60 --x-min -5 --x-max 5 --y-min -5 --y-max 5 --amplitude 1 --frequency 1 --sigma 2
```

Доступные значения `--function`:
- `"paraboloid"` — параболоид `z = x² + y²`
- `"wave"` — волна `z = A·sin(k·x)·cos(k·y)`
- `"ripple"` — воронка `z = sin(√(x²+y²))`
- `"saddle"` — седло `z = x² - y²`
- `"gaussian"` — гауссиан `z = A·exp(-(x²+y²)/σ²)`

### Шаг 4: Запустить скрипт

1. Нажмите кнопку ▶ **Run Script** (вверху текстового редактора) или `Alt+P`.
2. В 3D-вьюпорте появится новый объект `MathSurface_Mesh`.
3. Нажмите `Numpad 0` или `Numpad 5` для переключения видов.
4. Используйте средний клик мыши для вращения, колёсико — для масштабирования.

### Шаг 5: Проверить материал и освещение

1. Выберите объект `MathSurface_Mesh` (щёлкните на нём).
2. Справа в панели свойств перейдите в **Material Properties** (иконка шарика).
3. Убедитесь, что материал `Mat_<function>` применён автоматически.
4. При необходимости настройте `Base Color`, `Roughness`, `Metallic`.
5. Нажмите `Z` → выберите **Material Preview**.

### Шаг 6: Изменить функцию и сравнить

1. Запустите скрипт повторно с другим аргументом `--function`, например `saddle`.
2. Старый объект удалится, появится новый.
3. Сравните форму «седла» с «волной».

## Часть 2: Geometry Nodes (интерактивные ползунки)

Geometry Nodes позволяет изменять параметры **в реальном времени**, без повторного запуска скрипта.

### Шаг 1: Создать базовую сетку

1. Перейдите в **Layout** (вкладка вверху).
2. Удалите куб по умолчанию: выберите его → `Delete`.
3. Добавьте сетку: `Shift+A` → **Mesh** → **Grid**.
4. В появившейся панели внизу слева установите:
   - **X Subdivisions: 80**
   - **Y Subdivisions: 80**
   - **Size: 10** (или любой удобный)

### Шаг 2: Добавить Geometry Nodes Modifier

1. Выберите сетку (Grid).
2. Справа в панели свойств перейдите в **Modifier Properties** (иконка гаечного ключа, синяя).
3. Нажмите **Add Modifier** → **Geometry Nodes**.
4. Нажмите **New** — откроется редактор Geometry Nodes.

### Шаг 3: Создать граф нод для волны z = A·sin(k·x)·cos(k·y)

В редакторе Geometry Nodes создайте следующую цепочку нод:

```
[Group Input] → [Set Position] → [Group Output]
                     ↑
              [Vector: Combine XYZ]
                  X: Position.X
                  Y: Position.Y  
                  Z: [Math: Multiply]
                           ↑
                    [Math: Multiply] × A
                           ↑
                    [Math: Sin(k·X)] × [Math: Cos(k·Y)]
```

**Пошаговое создание:**

1. Нажмите `Shift+A` в редакторе Geometry Nodes → **Geometry** → **Set Position**.
2. Подключите `Group Input` → `Set Position (Geometry)` → `Group Output`.
3. Добавьте ноду позиции: `Shift+A` → **Input** → **Position**.
4. Добавьте `Shift+A` → **Utilities** → **Math** (Sin).
   - Установите операцию: **Sin**.
   - Подключите `Position.X` (компонент X) к входу Sin.
5. Добавьте ещё одну **Math** ноду (Cos), подключите `Position.Y`.
6. Добавьте **Math** (Multiply): умножьте Sin × Cos.
7. Добавьте **Math** (Multiply): умножьте на амплитуду A.
8. Добавьте `Shift+A` → **Utilities** → **Vector** → **Combine XYZ**:
   - X: `Position.X`
   - Y: `Position.Y`
   - Z: результат вычислений
9. Подключите **Combine XYZ** → **Set Position (Position)**.

### Шаг 4: Добавить ползунки (Group Input)

1. Выберите ноду **Group Input**.
2. Нажмите `N` в редакторе Geometry Nodes → перейдите на вкладку **Group**.
3. Нажмите `+` и добавьте:
   - **Amplitude** (Float, default: 1.0, min: 0, max: 5)
   - **Frequency** (Float, default: 1.0, min: 0.1, max: 10)
4. Подключите `Amplitude` к множителю, `Frequency` к входам Sin и Cos (через Math Multiply).

### Шаг 5: Использовать ползунки

1. Вернитесь в **Layout**.
2. В правой панели (N-panel, нажмите `N`) перейдите на вкладку **Item**.
3. Под названием объекта увидите ползунки **Amplitude** и **Frequency** — двигайте их!
4. Поверхность обновляется в реальном времени.

## Часть 3: Настройка освещения и рендера

### Шаг 1: Настроить свет

1. Выберите источник света (лампа в сцене по умолчанию).
2. В **Object Properties** справа:
   - Тип: **Sun** (параллельный свет, подходит для поверхностей)
   - Strength: **3–5**

### Шаг 2: Настроить камеру

1. Нажмите `Numpad 0` — вид из камеры.
2. Выберите камеру → `G` → `Z` → двигайте мышью вверх, чтобы поднять камеру.
3. Нажмите `R` → `X` → **−45** Enter, чтобы наклонить камеру вниз.

### Шаг 3: Рендер (скриншот)

1. В меню **Render** → **Render Image** (или `F12`).
2. Дождитесь завершения рендера.
3. **Image** → **Save As** — сохраните PNG для отчёта.

## Часть 4: Дополнительные настройки для красивого результата

### Colormap по высоте (цвет зависит от z)

1. Выберите объект → **Material Properties** → **New**.
2. В нодах материала добавьте:
   - `Shift+A` → **Input** → **Texture Coordinate** → **Generated**
   - Подключите Z-компоненту к **ColorRamp** (`Shift+A` → **Converter** → **ColorRamp`)
   - Из ColorRamp → **Base Color** шейдера
3. Настройте цвета в ColorRamp (например, синий → белый → красный).

### Wireframe (сетка поверх поверхности)

1. В **Viewport Overlays** (иконка двух кружков вверху) включите **Wireframe**.
2. В настройках Overlay установите прозрачность wireframe ~0.3.

## Советы и частые проблемы

| Проблема | Решение |
|---------|---------|
| Скрипт запустился, но объект не виден | Нажмите `Numpad .` для центровки вида на объект |
| Поверхность слишком «острая» | Уменьшите амплитуду или увеличьте разрешение |
| Geometry Nodes не реагирует | Проверьте, что все ноды правильно соединены |
| Ошибка в Python-скрипте | Откройте **Info** окно снизу — там будет описание ошибки |
| Blender зависает при рендере | Уменьшите RESOLUTION в скрипте до 30–40 |
