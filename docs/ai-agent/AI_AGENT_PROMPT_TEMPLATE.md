## Критерии успеха

Проект принят только если:

1. Запуск и демонстрация возможны без устных пояснений.
2. Артефакты (`.blend`, `.docx`, рендеры, docs) целостны и согласованы.
3. Нет внутренних противоречий между файлами.
4. Репозиторий чистый и готов к передаче преподавателю/студенту.
5. Есть рабочий пример прикладной задачи (pathfinding/оптимизация) на 3D-поверхности.
