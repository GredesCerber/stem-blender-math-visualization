## 📞 Если вы в затруднении

| Проблема | Решение |
|---|---|
| Blender не запускается | Используйте full path: `"C:\Program Files (x86)\Steam\steamapps\common\Blender\blender.exe"` |
| Python ошибка импорта | Убедитесь что вы в папке проекта: `cd C:\Users\...\stem-blender-math-visualization` |
| .blend файл не открывается | Используйте Blender 4.0+ или 3.6+ |
| Ползунки не работают | Откройте Properties → Modifier (🔧) → GeoNodes_Surface |
| Экспортировать не получается | Просто оставьте markdown — PDF может быть сделано позже |
