# Теория: Функции двух переменных и математические поверхности

## 1. Что такое функция двух переменных

Функция двух переменных — это правило, которое каждой паре чисел **(x, y)** из некоторой области D ставит в соответствие единственное число **z**:

```
z = f(x, y),  где (x, y) ∈ D ⊆ ℝ²
```

**Примеры из повседневной жизни:**
- Температура воздуха в зависимости от долготы и широты: `T = f(φ, λ)`
- Высота рельефа местности в зависимости от координат: `h = f(x, y)`
- Давление газа в зависимости от температуры и объёма: `P = f(T, V)`

---

## 2. График функции двух переменных — поверхность

График функции `z = f(x, y)` — это **поверхность** в трёхмерном пространстве: множество точек `(x, y, f(x, y))`.

> Если двигаться вдоль оси X или Y на некоторую величину и наблюдать, как меняется z — это называется **частная производная** и описывает «крутизну» поверхности в этом направлении.

---

## 3. Основные типы поверхностей

### 3.1 Параболоид вращения

```
z = x² + y²
```

- **Форма:** «чаша» или «воронка», симметричная относительно оси Z.
- **Минимум:** в точке `(0, 0, 0)`.
- **Применение:** фокусирующие антенны, зеркала телескопов, оптика.

![Параболоид: z = x²+y²]

### 3.2 Гиперболический параболоид (седло)

```
z = x² - y²
```

- **Форма:** седло: вогнутый вдоль X, выпуклый вдоль Y (или наоборот).
- **Особая точка:** `(0, 0, 0)` — «седловая точка» (не минимум и не максимум).
- **Применение:** геодезические купола, архитектурные конструкции.

### 3.3 Тригонометрическая волна

```
z = A · sin(k·x) · cos(k·y)
```

- **Параметры:**
  - `A` — амплитуда (высота «гребней»)
  - `k` — частота (количество «волн» на единицу длины)
- **Форма:** двумерная рябь, яйцеклеточная сетка.
- **Применение:** моделирование волн на воде, виброакустика.

### 3.4 Радиальная волна (воронка)

```
z = sin(√(x² + y²))
```

- **Форма:** концентрические волновые кольца вокруг начала координат.
- **r = √(x²+y²)** — расстояние от центра (полярный радиус).
- **Применение:** распространение волн от точечного источника.

### 3.5 Гауссова поверхность (колокол)

```
z = A · exp(-(x² + y²) / σ²)
```

- **Параметры:**
  - `A` — высота (пик распределения)
  - `σ` — «ширина» (стандартное отклонение)
- **Форма:** гладкий купол, убывающий к нулю вдали от центра.
- **Применение:** нормальное распределение в статистике, теория вероятностей.

### 3.6 Гиперболоид

```
z = √(1 + x² + y²)    или    z = √(x² + y² - 1)
```

- **Форма:** «рюмка» (однополостный) или «корсет» (двуполостный).
- **Применение:** охладительные башни электростанций, архитектура.

## 4. Дискретизация: от формулы к сетке

В Blender (и в компьютерной графике вообще) непрерывную поверхность нельзя изобразить напрямую — нужно **дискретизировать** её: взять конечный набор точек.

**Алгоритм:**
1. Задаём область: `x ∈ [x_min, x_max]`, `y ∈ [y_min, y_max]`.
2. Делим каждую ось на `N` равных шагов: `Δx = (x_max - x_min) / N`.
3. Для каждой пары `(i, j)` при `i, j = 0, 1, ..., N` вычисляем:
   ```
   x_ij = x_min + i · Δx
   y_ij = y_min + j · Δy
   z_ij = f(x_ij, y_ij)
   ```
4. Из точек `(x_ij, y_ij, z_ij)` строим сетку (mesh): квадраты → полигоны.

**Качество сетки:** чем больше N, тем плавнее поверхность, но тем медленнее работает. Для учебных целей достаточно `N = 40–100`.

---

## 5. Частные производные и крутизна

Частная производная по x показывает, как быстро меняется z при малом изменении x (при фиксированном y):

```
∂f/∂x = lim[Δx→0] (f(x+Δx, y) - f(x, y)) / Δx
```

**Применение в Blender:** нормали к поверхности (направление освещения, шейдинг) вычисляются именно через частные производные. Blender делает это автоматически по геометрии сетки.

---

## 6. Параметры и их влияние

| Параметр | Обозначение | Что меняет |
|----------|-------------|-----------|
| Амплитуда | A | Высоту «гребней» и «впадин» |
| Частота | k | Количество повторений на единицу длины |
| Масштаб | scale | Общий размер поверхности |
| Смещение | offset | Сдвиг поверхности по Z |
| Разрешение | N | Количество полигонов (качество) |
