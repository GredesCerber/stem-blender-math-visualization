# Интерактивная 3D-визуализация математических функций и поверхностей в Blender

> **Учебный STEM-проект**  
> Павлодарский педагогический университет  
> Направление: Информатика и математика

---

## О проекте

Проект демонстрирует, как с помощью **Blender** (Geometry Nodes и Python/bpy) можно строить интерактивные трёхмерные графики математических функций вида **z = f(x, y)**. Изменяя параметры (амплитуду, частоту, масштаб) прямо в интерфейсе Blender, студент наглядно видит, как меняется форма поверхности — волны, параболоид, седло, воронка.

Проект упакован как **методическое пособие**: всё, что нужно, чтобы с нуля запустить демонстрацию на занятии или провести учебное исследование.

## Быстрый старт

### Способ 1 — Python-скрипт (рекомендуется для начала)

1. Установите **Blender 3.6+** (https://www.blender.org/download/).
2. Откройте Blender → перейдите в **Scripting** (вкладка вверху).
3. Нажмите **Open**, выберите файл `scripts/generate_surface_mesh.py`.
4. Нажмите ▶ **Run Script**.
5. В 3D-вьюпорте появится поверхность `z = sin(x) * cos(y)`.
6. Измените формулу в коде (строка с `def f(x, y)`) → запустите снова.

### Способ 2 — Geometry Nodes (интерактивные ползунки)

Следуйте инструкции в [`docs/03_blender_guide.md`](docs/03_blender_guide.md), раздел **«Geometry Nodes: шаг за шагом»**.

## Реализованные функции

| Функция | Формула | Описание |
|---------|---------|----------|
| Параболоид | `z = x² + y²` | Параболическая чаша |
| Волна | `z = A·sin(k·x)·cos(k·y)` | Двумерная тригонометрическая волна |
| Воронка | `z = sin(√(x²+y²))` | Радиальная волна (синус от радиуса) |
| Седло | `z = x² - y²` | Гиперболический параболоид |
| Гауссиан | `z = A·e^(−(x²+y²)/σ²)` | Гауссова поверхность (колокол) |
