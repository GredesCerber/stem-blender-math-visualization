# STEM-проект: Интерактивная 3D-визуализация математических функций в Blender

> **«Сначала — зачем. Потом — как.»**

---

## Зачем это нужно?

Математика в школе и университете часто преподаётся через уравнения и двумерные графики. Студент видит формулу вроде **z = sin(x)·cos(y)** и должен мысленно представить, как она выглядит в пространстве. Большинство учеников не могут этого сделать — не потому что им не хватает умственных способностей, а потому что у них нет подходящего инструмента и достаточного опыта пространственного мышления.

Результат — **непонимание, потеря мотивации, ощущение «математика — это не моё»**.

Этот проект отвечает на вопрос: *«Что если дать ученику возможность буквально вертеть трёхмерный график руками, менять его параметры и немедленно видеть результат?»*

---

## Что это за проект?

**STEM-проект «Интерактивная 3D-визуализация математических функций и поверхностей в Blender»** — это:

- **учебная методичка** для учителей и студентов;
- **набор Python-скриптов** (Blender API `bpy`) для генерации 3D-поверхностей по формулам;
- **инструкция по Geometry Nodes** для интерактивного управления параметрами прямо в Blender;
- **серия экспериментов** — как изменение параметра влияет на форму поверхности;
- **документация на русском языке** с обоснованием актуальности и методическими рекомендациями.

Проект реализован в **Blender** (бесплатно, кроссплатформенно) и **VS Code** (редактор скриптов).

## Ключевые математические поверхности в проекте

| Функция | Название | Особенность |
|---|---|---|
| z = x² + y² | Параболоид | Классическая чаша |
| z = A·sin(kx)·cos(ky) | Рябь/волны | Параметры A и k |
| z = sin(√(x²+y²)) | Круговые волны | Радиальная симметрия |
| z = x² − y² | Седло (гиперболический параболоид) | Смена знака |
| z = e^(−(x²+y²)) | Гауссов колокол | Нормальное распределение |
